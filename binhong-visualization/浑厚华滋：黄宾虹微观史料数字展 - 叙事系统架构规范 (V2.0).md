# 浑厚华滋：黄宾虹微观史料数字展 - 叙事系统架构规范 (V2.0)

本规范定义了“环境压力/历史转折点”作为核心叙事引擎的架构，通过动态图层联动，将个人生平与大历史背景进行像素级对齐。

---

## 📐 一、 视觉架构：双层时间轴与潮汐背景

### 1.1 双层时间轴系统 (Dual-Layer Timeline)
* **上轴（时代波形图 - Pressure Wave）**
    * **形态**：非线性刻度，使用一条贯穿底部的 SVG 样条曲线（Spline）。
    * **逻辑**：曲线高度 $H$ 由该年份的 `pressure_index` 决定。转折点（如1937）形成波峰，稳定期形成波谷。
    * **交互**：悬停时显示“历史背景概览”，点击可触发深度视差视图。
* **下轴（生命线 - Life Track）**
    * **形态**：标准线性刻度尺，保持现有的编年导航功能。
    * **联动**：点击下轴任一刻度，上轴的波形图对应点同步高亮。

### 1.2 背景“潮汐”动效 (Atmospheric Tide)
* **视觉隐喻**：全局背景覆盖一层基于 Canvas 的动态粒子流。
* **动态参数**：
    * **速度**：`velocity = 1 + (pressure_index * 0.5)`。压力越大，光影流移动越快。
    * **色调**：由宣纸色（`#F4F1EA`）向炭灰色（`#4a4a4a`）过渡，利用 GSAP 平滑处理颜色向量。

## 🔄 二、 交互机制：专题视差与深度镜像

### 2.1 专题视差视图 (Era Reflection View)
当用户定位至关键年份时，触发“深度叙事模式”：
* **地图联动**：叠加半透明的“战火/避难路径图”矢量图层。
* **时代镜像卡片 (Era Reflection)**：
    * **外部环境**：提取该年度重大历史事件。
    * **个人响应**：描述在该环境下，黄宾虹艺术/生活的具体转向。
    * **规律洞察**：根据数据归纳出来的学术论断（如：“乱世迁徙路径分析：黄宾虹的避难选择与江南文脉的避险方向高度一致。”）。

### 2.2 专题命题式探索 (Thematic Exploration)
在界面顶部部署 `Dropdown` 菜单：
* **模式一 [动荡与隐遁]**：过滤战争转折年份，对比迁徙路径与战线关系。
* **模式二 [政治光谱]**：激活人物关系网，按政治属性聚类渲染，揭示生存策略。

## 🛠️ 三、 开发执行与数据规范 (Technical Specification)

### 3.1 数据结构规范 (Data Schema)
须构建两张关联表，通过 `year` 字段映射：
* **个人年谱表**：`{id, year, location, figures, content}`
* **时代背景表**：`{id, year, pressure_index, event_description, force_path_data}`

### 3.2 显隐控制开关 (Global Toggle)
* **简洁模式**：仅显示个人生平，背景潮汐关闭。
* **增强模式**：开启压力波形图、背景粒子流、时代镜像模块。

### 3.3 开发核心伪代码参考
```javascript
// 切换模式逻辑
function toggleBackgroundMode(enabled) {
    if (enabled) {
        Timeline.showPressureWave();
        BackgroundEngine.startParticleFlow();
        UIManager.showEraReflection(currentData);
    } else {
        Timeline.hidePressureWave();
        BackgroundEngine.stop();
        UIManager.hideEraReflection();
    }
}
```

## 四、 给各协作方的一键操作单

### 📋 给 @UI设计师 (成员C)

1. **色彩编码**：建立从“淡米色（平和）”到“炭灰/暗朱砂（动荡）”的映射表。
2. **图层设计**：地图需支持“多层级叠加”，确保底图、迁徙线、战火动态图、避难路径图的堆叠不影响可读性。
3. **镜像模块**：设计一种带有“印章”质感的视觉组件，用于容纳“时代镜像卡片”。

### 📋 给 @前端开发 (成员D)

1. **数据层**：实现 `HistoryContextService`，负责将 `EventSet` 与 `PersonalLog` 进行时间序列对齐。
2. **动效层**：使用 `GSAP` 实现背景粒子流的平滑变色，严禁硬切。
3. **图表层**：ECharts 中自定义 `visualMap` 来表现“压力曲线”的颜色过渡，为迁徙路线添加 `effectScatter` 模拟流向。
